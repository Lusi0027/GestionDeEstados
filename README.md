# Sistema de Gestión de Estados de solicitudes

**Descripción del proyecto**

Este proyecto consiste en una aplicación que permite gestionar solicitudes de clientes dentro de una empresa de servicios tecnólogicos.

Su propósito es brindar un control claro, ordenado y confiable sobre cada solicitud, permitiendo registrar, consultar y actualizar su estado de forma eficiente.

El sistema hace uso de enumeradores (enum) para manejar los estados de las solicitudes.

**¿Qué es enum?**

Es un tipo de dato especial en C# que permite definir un conjunto de valores constantes con nombre.

**Funcionalidades del sistema**
1. Registrar nuevas solicitudes de clientes.
2. Mostrar todas las solicitudes registradas.
3. Cambiar el estado de una solicitud existente.
4. Buscar una solicitud por ID.

**Ejecución del programa**

El sistema funciona mediante un menú de consola, donde el usuario puede seleccionar opciones para interactuar con las solicitudes.

El menú contiene las siguientes opciones:
1. Registrar solicitud
2. Mostrar solicitudes
3. Cambiar estado
4. Buscar solicitud por ID
5. Salir
